Notion API token (tap to set up) ▸

You only need to do this once. Your token is saved in this browser only and never sent anywhere except Notion.

How to get your token:
1. Open notion.so/profile/integrations in a new tab
2. Click New integration, give it any name, click Save
3. Copy the Internal Integration Secret (starts with secret_)
4. Go to your Reading List page → click ··· top right → Connections → connect your integration
5. Paste the secret below and click Save
