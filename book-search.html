Add a Book

Search for any book and add it to your Reading List.

Notion API token

You only need to do this once per device.

How to get your token:
1. Go to notion.so/profile/integrations
2. Click New integration
3. Give it a name (e.g. "Book Search"), select your workspace, click Save
4. Copy the Internal Integration Secret (starts with secret_)
5. Go to your Reading List page in Notion → click the ··· menu → Connect to → select your integration
6. Paste the secret below
